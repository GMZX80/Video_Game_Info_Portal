## Static Architecture

The public site remains a static GitHub Pages artifact. Search data is generated into `assets/data/generated/` and copied into `dist/`.

## Candidate Records

Candidate source assertions are searchable as `External source assertion`. They include source system, platform, evidence status, source URL, revision/permanent URL where relevant, and warning text that they require corroboration.
