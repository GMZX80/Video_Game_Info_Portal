# Manual Import Templates

Use this directory for reviewed, permitted manual imports that cannot be obtained from an official API.

Rules:

- Do not scrape MobyGames HTML.
- Do not paste long article text, screenshots, scans, binaries or other protected media.
- Keep each person credit as candidate/secondary unless corroborated by a permitted primary or reviewed source.
- Preserve source URLs, access notes and reviewer notes for every row.
